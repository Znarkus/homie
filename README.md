# homie
Central hub for controlling your home.

## Install
```
git clone https://github.com/Znarkus/homie.git
cd homie
```

Then install:
```
npm install
bower install
```

Create and edit the config file:
```
cp config.example.js config.js
nano config.js
```

## Run
```
node .
```

## Run forever
```
npm install -g forever
forever start .
```
